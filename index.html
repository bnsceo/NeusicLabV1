FAQ

Is NeusicWave free?

Yes. The demo is entirely free and includes the full suite. Save and cloud archiving unlock when you upgrade to Neusic Suite Pro.

Do I need to be online?

No. All audio engines run locally in your browser. The internet is only needed to load the page and, if you choose, to sync.

Can I use it on my phone?

Yes. Every tool is mobile-optimized — panels collapse and stack on smaller screens.

Do I need an API key?

No API key is required to create. A key only unlocks the optional Neusic Agent for AI mixing and arrangement help.

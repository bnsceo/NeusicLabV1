Core Features

Everything in one workflow

Six apps in one. No switching. No exporting between tools. Zero friction from idea to master.

🎚
MPC-Style Sampler

Load any audio file, auto-chop it with real transient detection, and play slices on 16 velocity-sensitive pads. Banks A–D give you 64 pads. Swing, note repeat, and 16 Levels mode included.

Sampling First
🎹
88-Key Piano Roll

Full 88-key scrollable grid with quantised note drawing, drag-to-move, resize, velocity lane editing, note properties popover, playhead follow, and real-time MIDI note preview.

Full Range
🥁
Drum Machine + Sequencer

16-step sequencer for 8 tracks. Synthesised one-shot drums (no samples needed). Live pad performance with a look-ahead scheduler for rock-solid timing.

Beat Making
🎤
Studio Recording

One-tap mic recording with real VU metering, metronome, count-in, overdub, and loop modes. Records directly to the timeline as a proper audio clip.

Pro Quality
🎚️
Real FX Rack

Per-track insert chain: Reverb (convolver), Delay (with feedback), Chorus (LFO), EQ (high-shelf tilt), Compressor. All wired as real Web Audio nodes — not just UI.

Live DSP
🎛️
Full Mixer

Per-track faders, stereo panning, mute/solo with real audio graph management, real-time VU meters, and master gain control. All draggable on touch and mouse.

Real Audio

Automation

Draw volume, pan, and filter cutoff automation curves that actually move AudioParams during playback using linearRampToValueAtTime — proper DAW automation, not fake.

Scheduled

Sample Browser

Import files or scan an entire folder. Category filter (Drums, Loops, Vocals, Bass…), live search, BPM and key detection from filename, preview on tap, drag-to-track.

Web MIDI API — connect any hardware keyboard or controller. Live note-on/off to piano roll, drum pad triggering on channel 10, CC mapping with MIDI Learn, pitch bend.

v3 Changelog

What changed in this build

Major upgrades compared to the previous version — everything you just shipped.

1
Refined dark neumorphic UI — entire visual overhaul

Deeper bg (#1c1c2e), polished gradients on track headers, clip backdrop-blur, glow overlays on the overview bar, new playhead shape (rounded rect cap), improved scrollbar styling, and a glass-effect center column background.

2
MPC 4×4 pad grid with Banks, Swing, Note Repeat, 16 Levels

The sampler now has a real MPC-style pad grid (A/B/C/D banks = 64 pads), velocity-sensitive hit detection, swing timing on even pads, note-repeat hold, and 16 Levels velocity ladder.

3
Piano Roll: velocity lane drag editing + note popover

Drag the velocity lane to edit individual or range-selected note velocities. Right-click any note to open a popover to edit beat, length, velocity, preview, or delete — precise editing without leaving the roll.

4
Piano Roll: real synth note preview + playhead follow

Every note click plays a real oscillator voice through the track's FX chain. Playhead auto-scrolls the grid during playback (⏩ Follow toggle). "⇔ Sync zoom" matches PR zoom to the timeline.

5
Web MIDI device input with MIDI Learn + MIDI record

Connect hardware keyboards and drum pads via Web MIDI API. Note-on/off routes to piano roll or drum pads. MIDI Learn maps CC knobs to BPM or volume. MIDI record captures real-time performance with accurate note lengths.

6
Sample Browser with folder scan, categories, drag-to-track

New Browser tab lets you import files or scan an entire folder (File System Access API on Chrome/Edge). Filters by category, live search, preview on tap, and drag any sample directly onto the timeline at any beat position.

7
Clip editing: fade, gain, pitch, stretch, crossfade, bounce

Context menu now has 6 new clip operations. All non-destructive metadata on the clip. Bounce to clip uses OfflineAudioContext to bake pitch+stretch into a new buffer inline.

8
Project save/load (JSON) + WAV export

💾 Save the full project state to a JSON file. 📂 Load it back — tracks, clips, automation, FX, BPM all restored. ⬇ Export a full stereo mixdown as WAV using OfflineAudioContext offline rendering.

9
Track header shows type label + clip shows beat/length

Each track header now shows a third line — the track type (AUDIO / BEAT / MIDI). Each clip has a sub-label showing its start beat and length in beats for precise arrangement reading.

10
Wider track headers, zoom readout, CSS grid header layout

Track header width increased to 152px (168px on desktop). The header uses CSS grid (icon / meta / controls) for pixel-perfect alignment. The zoom group now shows a live readout ("1.0×"). Timeline metrics sync on every resize.

What's Missing · Roadmap

What comes next

Honest assessment of the current gaps and what's planned to close them.

📍
MIDI clip → timeline sync (clip position gates note playback)

Synth notes schedule from beat 0 regardless of where the MIDI clip sits on the timeline. If your Piano clip starts at bar 5, notes should be silent until bar 5.

Next
💾
Project persistence via localStorage / IndexedDB

Refresh the page and the project is gone. LocalStorage for project JSON plus IndexedDB for audio buffers would make Neusic session-safe.

Next
🔁
Loop region on the timeline

No way to set an A/B loop range yet. Essential for beat-making — loop 2 bars while tweaking the drum pattern.

Next
🎵
Polyphonic synth engine per MIDI track

Each MIDI track should have its own synth with selectable waveform, ADSR envelope, and filter — not just a shared global oscillator. Think mini-Massive per track.

Soon
📱
Multi-touch gesture hardening on mobile

Pinch-zoom and clip drag/resize use touch handlers that need a proper test pass on real devices. Long-press for context menu instead of right-click on phones.

Soon
🔊
Master bus limiter + loudness metering

The master output has no limiting or LUFS measurement. For release-ready tracks you need a true peak limiter and integrated loudness meter on the master.

Soon
🤝
Cloud sync + real-time collaboration

Projects live in the browser only. Real producers share sessions. Cloud save (Supabase/Firebase) and a link-to-collaborate workflow would unlock the full PRD vision.

Later
🤖
AI Music Assistant

The PRD calls for an AI assistant. Generate chord progressions, suggest drum patterns, analyse the mix for EQ problems, auto-master. This is the moat no other DAW has.
